## Project Overview

This project builds an interpretable AI-driven therapeutic target prioritization framework for lupus nephritis.

The analysis integrates outputs from:

- Human single-cell RNA-seq target discovery
- Mouse lupus nephritis validation
- Human spatial transcriptomics validation
- CellChat cell-cell communication analysis

The goal was to prioritize macrophage-associated therapeutic targets using multi-modal biological evidence rather than relying on differential expression alone.

## Key Findings

The weighted AI model prioritized:

1. **C5AR1**
2. **LILRB2**
3. **PILRA**
4. **CSF1R**

C5AR1 emerged as the strongest candidate due to high macrophage specificity, strong spatial validation, macrophage correlation, and cross-species support.
